# Regs-cipe Cookbook
A small community for home chefs sharing their best recipes, built with NodeJS.

## Features:

- Add recipe to bookmark.
- Authentication.
- Comment on recipes.
- Ingredient shopping list.
- Rate recipes.
- Reply to comments.
- Search recipes by words/tags.
- Submit blogs.
- Submit recipes.
- Upload images to cloudinary.
- View bookmarks.
- View recipes.
- View relevant recipes.
